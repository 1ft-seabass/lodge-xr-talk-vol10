Meta Quest 3 や XREAL Air といった xR デバイスで xR + AI(ChatGPT) + IoT 連携するプロトタイプを、体験できる展示です。"声でうごかす" をテーマに xR とChatGPT が会話できたり、IoT 操作ができます。
